js-stack
========

Standard JS SPA stack we use at netguru

## Installation

Add this line to your application's Gemfile:

    gem 'js_stack'

And then execute:

    $ bundle

## Usage

Modify `application.coffee` (you have to be using asset pipeline):

``` coffeescript
#= require js_stack
```
